    * Change password
## Dependencies
* Visual studio
* Xamarin Forms
* ASP.NET
* Sql Server

## How to start project

In visual studio, you just click Start button to create android emulator for run app.
To start ASP.NET you will start similar above by click Debug button or publish project.
